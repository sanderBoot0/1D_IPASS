Detailed Description

Class for communication throug the HC-12 chip.

With this class you can send a byte from one to another chip.
It is good practice to make use of the subclasses HC_12_uno and HC_12_due.
The subclasses automaticly assign the right wait_times for the serial data.
You can use this class instead of the HC_12_due class; The default variable is for the Arduino Due.

Constructor for a HC-12 interface.

This constructor creates an interface so you can communicate with the HC-12 chip.
Note that the name of the class is HC_12, and not HC-12 (The chip name).

Parameters
triggera type of pin_out, needs to be constructed in main code
echoa type of pin_in, needs to be constructed in main code
wait_timethis parameter is different depending on the Arduino type. It is 80 for the Uno, and 100 for the Due.
