hwlib
C++ OO-style close-to-the-hardware library
Analog interfaces

The (abstract) types adc and dac are used to read and write analog values.


adc

The hwlib::adc adc is an abstract class that provides an interface to an Analog-to-Digital (ADC) converter. Targets that have an ADC, and ADC peripheral chips, provide this interface.

attrubutes and functions meaning
adc_value_type the type returned by read()
adc_n_bits the number of significant bits in the result returned by read()
read() returns the adc value
refresh() makes sure the value returned by read() is up-to-date

An adc is buffered: a call to refresh() is needed to make sure that the value returned by read is up-to-date.


dac

The hwlib::dac dac is an abstract class that provides an interface to an Digital-to-Analog (DAC) converter. Targets that have a DAC, and DAC peripheral chips, provide this interface.

attrubutes and functions meaning
dac_value_type the type accepted by write()
dac_n_bits the number of significant bits in avlaue written by write()
write(v) writes the value v to the DAC
flush() makes sure the value written appears on the output

An dac is bufffered: a call to flush() is needed to make sure that a value that was written appears at the analog output.
